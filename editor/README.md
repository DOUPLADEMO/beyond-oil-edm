# Beyond Oil Hungary – Hírlevél sablon

## Fájlok

| Fájl | Leírás |
|---|---|
| `editor.html` | **Szerkesztő** — ezt nyissa meg szerkesztéshez |
| `index.html` | A kész, végleges hírlevél |
| `assets/` | Képek és logók ide kerülnek |

## Szükséges képek az `assets/` mappában

| Fájlnév | Leírás |
|---|---|
| `logo-white.png` | Beyond Oil fehér logó |
| `product-pack.png` | Termékfotó (csomagolás) |
| `rating-stars.png` | Testimonials háttérkép |
| `oil-bg.jpeg` | Science szekció háttérkép |

## Szerkesztés lépései

1. Nyissa meg az `editor.html` fájlt böngészőben (Chrome vagy Edge ajánlott)
2. Kattintson bármelyik szövegre — zöld keret jelzi a szerkeszthető mezőket
3. Módosítsa a tartalmat közvetlenül a designban
4. Kattintson a **"Mentés (index.html letöltése)"** gombra a képernyő alján
5. A letöltött `index.html` a végleges, szerkesztőelem-mentes fájl — ezt töltse fel

## GitHub Pages

Ha GitHub Pages-en szeretné hosztolni:
1. Töltse fel az összes fájlt a repo gyökerébe vagy `docs/` mappába
2. Kapcsolja be a Pages-t a repo Settings > Pages menüpontban
3. Az `index.html` lesz az élő verzió
